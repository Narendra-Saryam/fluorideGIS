# Nagpur GIS Next.js Demo

This repository is a demo Next.js app that shows a Leaflet-based GIS map for Nagpur district with sample fluoride values and a server API route stub for Gemini AI risk reasoning.

Important: The GeoJSON provided is synthetic sample data. Replace with official polygon GeoJSON for production.

How to run:

1. Install:
   npm install

2. Add a .env.local (optional) with GEMINI_API_KEY (if you plan to integrate real Gemini).

3. Run:
   npm run dev

Open http://localhost:3000
